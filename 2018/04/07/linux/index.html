基础用户操作

1.useradd

作用添加用户

参数 功能
-s 指定用户shell,具体不清楚，只是之前配置sftp用户时在网上查据说用于禁止用户ssh登陆，设-s /bin/false
-g 决定用户所在的组
-p 决定用户密码
-d 家目录，用户的根目录

以上配置也可在创建用户后用usermod添加或更改
密码还可以用passwd

删除用户userdel userName

2.groupadd

添加组
groupadd groupName
groups查看组中所有用户
groups userName查看用户所属的组名

删除组groupdel groupname

3.chown

更改目录所属用户
chown option user(group) file(dir)
