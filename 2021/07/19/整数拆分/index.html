最大公约数

求组合数 C(2n, 1), C(2n, 3), C(2n, 5) … C(2n, 2n - 1)的最大公约数

这道题考的是数学，代码只需要些一两行

第一步：证最大公约数必是 2 的指数幂形式

用二项式展开

(1+1)2n=C2n0+C2n1+C2n2+...+C2n2n=22n(1 + 1) ^ {2n} = C_{2n}^0 + C_{2n}^1 + C_{2n}^2 + ... + C_{2n}^{2n} = 2^{2n}(1+1)2n=C2n0​+C2n1​+C2n2​+...+C2n2n​=22n

(1−1)2n=C2n0−C2n1+C2n2+...−C2n2n−1+C2n2n=02n(1 - 1) ^ {2n} = C_{2n}^0 - C_{2n}^1 + C_{2n}^2 + ... - C_{2n}^{2n - 1} + C_{2n}^{2n} = 0^{2n}(1−1)2n=C2n0​−C2n1​+C2n2​+...−C2n2n−1​+C2n2n​=02n

联立上式有

C2n1+C2n3+...+C2n2n−1=22n−1C_{2n}^1+C_{2n}^3+...+C_{2n}^{2n-1} = 2^{2n - 1}C2n1​+C2n3​+...+C2n2n−1​=22n−1

若上述组合数存在最大公约数 a 那么22n−12^{2n - 1}22n−1一定可以被 a 整除，显然 a 是 2 的指数幂形式或者 1(2 的 0 次方)

因此上述所有二项式都可以化为k2xk2^xk2x的形式。
